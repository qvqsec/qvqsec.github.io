#### CVE-2025-2304
According to Tenable's report, when a user changes their password, the `updated_ajax` method of the `UsersController` is called. Here's the vulnerable code.


I'll look up the binary on [GTFOBins](https://gtfobins.org/) and find that I can create a ruby file that spawns a shell, then use facter with sudo privileges to gain a root shell.
